# Prompt Engineering Best Practices

## Principle 1: Explicit Context

Bad:
Write a script about AI.

Good:
Create a 60-second LinkedIn video script explaining how AI improves university admissions.

---

## Principle 2: Audience Definition

Always specify:

- Audience
- Goal
- Platform
- Tone

---

## Principle 3: Output Structure

Always define:

- Hook
- Body
- CTA

---

## Principle 4: Constraint Definition

Specify:

- Duration
- Length
- Style
- Format

---

## Principle 5: Machine Readability

Prefer structured outputs:

- Markdown
- JSON
- Tables
