# Cross-Platform Consistency Guide

Supported Models

- GPT-4
- Claude
- Flowise Pipelines
- API Based LLMs

## Rules

1. Avoid model-specific syntax.

2. Use explicit instructions.

3. Define output format.

4. Define constraints.

5. Use JSON schemas when possible.

6. Avoid vague adjectives.

7. Use deterministic wording.

---

## Example

Instead of:

"Make it good."

Use:

"Generate a professional 60-second LinkedIn video script for university administrators."
